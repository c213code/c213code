## 📱 Featured Project

### HalalScanner
> AI-powered iOS app that scans food products and determines their halal status

- 📷 Camera scanning with **AVFoundation**
- 🤖 Product recognition via **Roboflow ML model**
- 🔐 Authentication with **Firebase Auth**
- 💾 Data storage with **Firestore**
- 🏗 Architecture: **MVVM + Coordinator**
- ⚡️ Reactive bindings with **Combine**
- 📐 Fully **programmatic UI** with NSLayoutConstraint
